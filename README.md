# Unibo Pet Therapy

## Avvio applicativo
- Avviare XAMPP
- Creare il database con lo script [qui](./db/crea_db.sql)
- Se necessario, cambiare le credenziali per l'accesso al DB [qui](./www/bootstrap.php)
- Se il repo è stato clonato in `XAMPP/htdocs/`, allora potete accedere direttamente all'app web col seguente [link](http://progetto-web/www/index.php)
